## 🎯 Key Design Decisions

### Data Flow Model
```python
User Input
    ↓
SystemInput (validation)
    ↓
DesignBuilder (calculations)
    ↓
DesignObject (single source of truth)
    ↓
Multiple Renderers (SLD, GA, BOM)
    ↓
PDF Report (combines all)
```

### No Duplicate Logic
- **Calculations only in:** `engine/calculations.py`
- **Rendering only in:** `renderers/*.py`
- **Constants only in:** `config/constants.py`
- **All outputs from:** Single `DesignObject`

### Session State Caching
```python
Generate Button → Cache DesignObject
Download Button → Reuse DesignObject (no recalculation)
```
